# CurrencyBank
Консольное приложение на C# - симулятор банковского счета с авторизацией и конвертацией валют.

## Возможности
- Вход по логину/ паролю (логин - Nikchaprestol пароль - 0001)
- Пополнение счета в разных валютах
- Снятие средств (с проверкой баланса)
- Просмотр курса валют
- Конвертация рублей в другие валюты
